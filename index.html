Sobre o Sistema

O Monitor de Enchentes do Acre é um sistema simplificado para acompanhamento dos níveis dos rios no estado. Desenvolvido para facilitar o acesso às informações sobre a situação hídrica dos principais rios acreanos.

Como funcionam os alertas
1
Monitoramento Contínuo

Os níveis dos rios são monitorados continuamente através de estações fluviométricas.

2
Análise de Dados

Os dados são analisados em relação aos níveis históricos e cotas de alerta e inundação.

3
Emissão de Alertas

Com base na análise, são emitidos alertas quando necessário, classificados por nível de risco.
